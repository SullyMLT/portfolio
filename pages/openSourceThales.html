Aperçu du projet

À propos du stage

Durant mon stage chez Thales, j'ai eu l'oportunité de travailler sur l'amélioration d'un projet open source et d'ajouter de nouvelles fonctionnalités. Ce projet est un éditeur graphique de prototypage d'IHMs (Interfaces Homme-Machine) nommé Pyramid.

L'ajout des fonctionnalités que j'ai pu développer et pousser sur GitHub a été une expérience enrichissante. J'ai appris à utiliser Git et GitHub pour gérer le code source, collaborer avec d'autres développeurs et suivre les modifications apportées au projet.

Cette expérience m'a permis de maîtriser les bonnes pratiques de développement open source, telles que la rédaction de documentation claire, la gestion des issues et des pull requests, ainsi que la communication avec la communauté de développeurs.
